### ArrayList, Vector 
- 기존 배열의 크기 제한에 따른 문제를 해결한 것으로 자료구조 중에 배열의 개념을 구현 
- Vector는 쓰레드에서 동기화를 기본적으로 수행하고 ArrayList는 개발자가 추가해야 합니다.
